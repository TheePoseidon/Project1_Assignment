# Q2: Control Flow and repetition

## Example Sample Input/output

```bash
====Menu====
1. Add two numbers
2. Check even or odd
3. Print Numbers 1-10 (skip 5)
4. Exit
Enter your choice: 1

1st number: 25
2nd number: 15
Sum = 40
```
## Explanation of program logic

The program uses a shile loop to continuously display a menu until the user chooses to exit.
A switch statement handles the user's menu selection.
A conditional statement if-else is used in option 2 to check for even or odd.
Option 3 demonstrates the use of a for loop with continue to skip the number 5. 
The program terminates when the user selects option 4
