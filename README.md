# C programming formative assignment

## Project Overview
This repository contains C programs demonstrating various programming concepts such as functions, recursion, and basic I/O operations. The programs are organized by functionality and showcase good coding practices in C.

---

## Features
- **Simple I/O Operations**: Programs that take user input and produce output.  
- **Function Usage**: Programs broken into multiple functions for modularity and code reuse.  
- **Recursion**: Includes recursive functions to solve problems like factorial calculation.  
- **Error Handling**: Examples of handling basic syntax and logical errors.  
---

# Intergration

Control flow, functions, and embedded systems concepts work together to create reliable real-world applications. Control flow allows the system to make decisions and repeat tasks, functions organize code into reusable modules, and embedded systems combine software and hardware to interact with the physical environment. For example, in a smart temperature monitoring system, functions read sensor data, control flow determines whether the temperature is too high, and the embedded hardware activates an LED or cooling system based on the decision.
